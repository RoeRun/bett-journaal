# Bett Journaal - Stichting Chrono

Interactieve, cloud-based webapp voor Stichting Chrono: een visueel Bett-journaal.

## Setup

1. Maak een Supabase project aan op https://supabase.com
2. Kopieer `.env.example` naar `.env.local` en vul de Supabase credentials in
3. Deploy naar Vercel (of gebruik GitHub Codespaces voor development)

## Development

Voor lokale development (als je Node.js hebt):
```bash
npm install
npm run dev
```

Voor cloud-based development:
- Gebruik GitHub Codespaces
- Of push naar GitHub en gebruik Vercel's build systeem

## Deployment

Deploy naar Vercel:
1. Push code naar GitHub
2. Import project in Vercel
3. Voeg environment variables toe
4. Deploy!
